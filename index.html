Windows GUI Automation with Python

pywinauto is a set of python modules to automate the Microsoft Windows GUI. At itâ€™s simplest it allows you to send mouse and keyboard actions to windows dialogs and controls, but it also supports more complex actions.

It is simple and the resulting scripts are very readable. How simple?

app.UntitledNotepad.MenuSelect("Help->About Notepad")
app.AboutNotepad.OK.Click()
app.UntitledNotepad.Edit.TypeKeys ("pywinauto Works!", with_spaces = True)

Setup

  • Install pyWin32 extensions (no need for Active Python except 3.4 64-bit)
  • Download latest version
  • Just unpack and run python setup.py install

Supported controls

  • Native Windows controls (full support through Win32 API)
  • .NET Windows Forms (partial support through Win32 API, some basic controls only)
